Вариант 1: localtunnel (бесплатный, без регистрации)
1. Установите Node.js и npm (если ещё не установлены):
- Скачайте с https://nodejs.org/
- Установите
2. Установите localtunnel:
npm install -g localtunnel
3. Запустите Django сервер:
cd /Users/rustamismagilov/Desktop/ready
python3 manage.py runserver
4. В другом терминале запустите туннель:
lt --port 8000
5. Скопируйте вывод (пример: https://fancy-bison-123.loca.lt)
6. Обновите .env файл:
MEDIA_BASE_URL=https://fancy-bison-123.loca.lt/media/
7. Перезапустите Django сервер:
# Остановите старый сервер (Ctrl+C)
python3 manage.py runserver
