### Инструкции по использованию

#### Экспорт в Excel
1. Выберите книги в каталоге
2. Действие → "Экспортировать выбранные в Excel"
3. Скачается файл `books_export.xlsx`

#### Экспорт в шаблон Ozon
1. Загрузите шаблон: Админка → "Шаблоны Ozon" → "Добавить"
2. Выберите книги в каталоге
3. Действие → "Экспортировать выбранные в шаблон Ozon"
4. Скачается заполненный шаблон

#### Импорт книг из JSON
```bash
python import_books.py
```

### Известные ограничения

- Поиск по ISBN работает только по точному совпадению
- Только один шаблон Ozon может быть активным одновременно
- Шаблон Ozon должен быть в формате .xlsx (не .numbers)
- Экспорт работает только для листа "Шаблон" в файле Ozon
